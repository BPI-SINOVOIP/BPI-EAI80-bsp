Overview
the usb HID application is a simple demonstration program based on the ugelis SDK,It is emuerted as a HID-compliant device , A PC application can be used to exchange data with the device.
Run the example
1. Plug-in the device ,which is running HID generic example into the PC. A HID-compliant mouse is enumerated in the Device Manager.
2. Run the test tool,if data is sent ,the same data  will  be received
