Overview
the usb HID mouse application is a simple demonstration program based on the ugelis SDK,It is emuerted as a mouse, Users can see the mouse arrow moving on the PC screen according in a rectangular rotation.
Run the example
1. Plug-in the device ,which is running HID mouse example, into the pc. A HID-compliant mouse is enumerated in the Device Manager.
2. The mouse arrow is moving on PC screen in the rectangular rotation.
